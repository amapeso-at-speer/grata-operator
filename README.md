## Setup

- For environment setup, please refer to [this guide](https://reactnative-archive-august-2023.netlify.app/docs/0.67/environment-setup).

## Running the Application

```bash
npm install

# For iOS
cd ios
pod install
cd ../

npm run start

# In a new terminal:
# For Android
npm run android

# For iOS
npm run ios
```
